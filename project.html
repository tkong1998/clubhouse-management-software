About the project.

This is the ISOM3320 individual final project done by KONG, Tung (Monique).

This is a booking system for the HandShake Residence Club House. Only staff (normal staff and managers) can access to the system (sign in). Everytime a member wants to make a reservation, s/he will call the center through the club house hotline. The staff will help them check the availability of the corresponding facility at the date and time provided by the member. If available, then the reservation is valid. If not, then the reservation is not valid, and the member needs to make another reservation. Staff can also modify the status of each reservation (e.g., Booked, Checked In, Checked Out, or Cancelled) under different circumstances. On top of the normal functions of staff, managers can perform more advanced functions, such as see the list of all reservations, see the 3 most popular facilities, see the utilization rate of each facility.

Check out more about the project on my Github repo!
